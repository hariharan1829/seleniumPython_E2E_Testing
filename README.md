# Selenium Python E2E Testing

This project demonstrates **End-to-End (E2E)** automation testing using **Selenium WebDriver** and **Pytest** in Python.  
It follows the **Page Object Model (POM)** design pattern to maintain clean, reusable, and scalable test scripts.

----------------------------------------------------------------------------------

## How to Run Tests

### Clone the repository

git clone https://github.com/hariharan1829/seleniumPython_E2E_Testing.git

pytest -m smoke --browser_name firefox --html=reports/report.html

------------------------------------------------------------------------------------

## Key Features     
✅ Page Object Model (POM) implementation

✅ Reusable utility functions for browser handling

✅ Custom Pytest fixtures for setup and teardown

✅ Dynamic web element handling

✅ HTML reporting using pytest-html
